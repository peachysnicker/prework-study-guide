HTML

  • DOCTYPE is a declaration that tells the browser what document type it should expect when it is opened (not case-sensitive).
  • HTML tag represents the root (start) of our HTML document with the language specified.
  • The head element contains information about the webpage.
  • The title element defines the title for a webpage. It is required and extremely important for Search Engine Optimization (SEO).
  • The body element represents the visible content shown to the user.
  • The style element defines the CSS styles associated with a website.
  • The base element defines the base URL for a webpage.
  • The link element connects an extername resource for the HTML document.
  • The meta element defines metadata such as the character set, description, keywords, author, and viewport.
  • The script element embeds client-side scripts in the HTML document.

CSS

  • The rel attribute specifies the relationship between the current document and the linked document resource.
  • The href attribute specifies the location (URL) of the external resource. This attribute uses a relative path.
  • Margin indicates how much space we want outside of an element.
  • Padding property adds space around the content inside an element. Not visible on a page but adds a buffer.
  • CSS Box Model: Every HTML element is a box with margins, borders, padding and content.
  • CSS Rule: Selector {property: value; property: value;} Selector is followed by a declaration.

Git

  • git status: checks wat branch we are currently on.
  • git checkout: checkout -b branch name: creates a new branch and switches to it.
  • git add: add a fike as it looks now to your next commit (stage).
  • git reset: unstage a file while retaining the changes in a working directory.
  • git init: intialize an existing directory as a git repository.
  • git clone [url]: retrievce an entire repository from a hosted location via URL.
  • git branch: list your branches. a* will appear next to the currently active branch.

JavaScript

  • Var: Variable is a named container that allows us to store data in our code.
  • = is the assignment operator to assign the specific data we want stored in our variable.
  • === operator checks if two values are equal and returns a Boolean result true, if values are not equal will return false.
  • Control flow is the order in which a comouter executes code in a script. JavaScript works from top to bottom.
  • An if statement is a way of introducing decision-making into our code. The code only runs if the statement meets a condition or is true.
  • An array is a single variable that is used to hold a group of data.
